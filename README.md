# dotfiles

configuration of my work environment.

for install this, in you HOME dir:
``` shell
git clone git://github.com/cawcaw/dotfiles
dotfiles/install
```

Install Vundle
``` shell
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle 
```

start VIM and:
``` vim
:BundleInstall
```
